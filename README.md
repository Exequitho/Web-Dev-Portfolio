Repositori ini berisi kumpulan simulasi aplikasi web yang dikembangkan secara spesifik untuk memenuhi standar uji kompetensi **Badan Nasional Sertifikasi Profesi (BNSP)** untuk skema **Asisten Web Developer / Junior Web Programmer**. 

Proyek ini dibangun menggunakan arsitektur Vanilla JavaScript dan Tailwind CSS guna mendemonstrasikan pemahaman fundamental logika pemrograman tanpa ketergantungan penuh pada *framework* berat.

---

## 🎯 Matriks Pemenuhan Kompetensi (SKKNI)

Portofolio ini dirancang untuk membuktikan 6 unit kompetensi utama:

1. **Mengimplementasikan User Interface:** Desain responsif (*mobile-first*), interaktif, dan menerapkan prinsip *Glassmorphism*.
2. **Menerapkan Perintah Eksekusi Berbasis Teks/Grafik:** Manipulasi DOM tingkat lanjut, pemformatan angka/mata uang dinamis, dan kalkulasi *real-time*.
3. **Menyusun Sumber Daya Pemrograman secara Rapi:** Penulisan kode terstruktur, modular, dan arsitektur *Clean Code*.
4. **Menulis Kode Sesuai Guidelines & Best Practices:** Validasi *client-side* ketat, penamaan *camelCase*, penggunaan *Custom Alert/Modal* modern.
5. **Mengimplementasikan Pemrograman Terstruktur:** Manajemen *state* kompleks, penggunaan *Nested Loops*, dan algoritma percabangan.
6. **Menggunakan Library / Pre-Existing Component:** Integrasi fungsional dengan Tailwind CSS (UI), FontAwesome (Icon), dan Chart.js (Visualisasi Data).

---

## 💻 Showcase Aplikasi

Repositori ini mencakup 4 modul aplikasi utama yang dapat diakses melalui satu halaman *dashboard* portofolio sentral:

### 1. Sistem Inventaris Core (Fundamental CRUD)
* **Fokus:** Opeasi *Create, Read, Update, Delete* penuh berbasis *Local Storage*.
* **Fitur Spesial:** Validasi form berlapis anti-null dan pencegahan input nilai negatif.

### 2. ArusKas App (Logika Kalkulasi Terstruktur)
* **Fokus:** Perhitungan matematika dasar dan format data grafis.
* **Fitur Spesial:** Rekapitulasi otomatis pemasukan/pengeluaran dan algoritma *sorting array* untuk menampilkan transaksi terbaru.

### 3. Pro-Manage Dashboard (Integrasi Library & Array Filter)
* **Fokus:** Manipulasi objek waktu (*Date*) dan visualisasi data.
* **Fitur Spesial:** Visualisasi *pie chart* interaktif (*Chart.js*), filter pencarian *real-time*, dan deteksi status *overdue* otomatis.

### 4. SeatReserve Pro (Manipulasi DOM & State Lanjutan)
* **Fokus:** *Dynamic element generation* dan kalkulasi harga multi-tier.
* **Fitur Spesial:** *Nested-loops* untuk membangun matriks kursi interaktif dan sinkronisasi *state* belanja secara *live*.

## 🚀 Cara Menjalankan Proyek Secara Lokal

Karena proyek ini berjalan di sisi *client-side* murni, tidak diperlukan instalasi *server* lokal atau dependensi *Node.js* yang rumit.

1.  **Clone repositori ini:**
    ```bash
    git clone [https://github.com/exequitho/Web-Dev-Portfolio.git](https://github.com/exequitho/Web-Dev-Portfolio.git)
    ```
2.  Buka folder proyek yang baru saja diunduh.
3.  Klik ganda pada file `index.html` untuk membukanya langsung di *browser* pilihan Anda (Chrome/Firefox/Edge direkomendasikan).
4.  *(Opsional)* Gunakan ekstensi **Live Server** di VS Code untuk pengalaman *hot-reload*.
